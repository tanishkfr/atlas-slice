# Atlas

**Principles under pressure.**

Atlas is a research-through-design instrument for making interaction-design reasoning inspectable. A visitor writes one provisional principle, carries it through three distant cases, and preserves every place the principle holds, is refined, or fractures.

The result is a **stress trace**: a visible revision lineage, not an answer generated by the system.

## The interaction

**Claim → pressure → expose an assumption → hold, refine, or fracture → inspect the lineage.**

Progress is saved locally so the trace survives navigation and reloads. A completed trace can be copied or downloaded.

The supporting corpus contains 33 single-authored interaction-design arguments organized into four editorial reasoning patterns. The patterns informed the instrument; they are not presented as a validated taxonomy or as empirical proof.

## Run

~~~sh
npm install
npm run dev
~~~

~~~
