# Atlas

**Principles under pressure.**

Atlas is a research-through-design investigation into transferable interaction-design reasoning. It asks a visitor to state one provisional principle, carry it through distant cases, and preserve every place that principle holds, fractures, or changes.

The result is a **stress trace**, not an answer generated by the system.

## Interaction

**Claim → pressure → expose assumption → hold or fracture → revise → inspect lineage.**

The supporting corpus contains evidence-backed interaction cases. It earns trust in the method, while the stress trace demonstrates transfer on open ground.

## Run

```sh
npm install
npm run dev
```

Verification:

```sh
npm run build
npm run lint
```
